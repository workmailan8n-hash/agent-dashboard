# Architecture Decision Records

Append-only log of architectural decisions. Each ADR is short (≤300 words), dated, and marked with status.

**Status values:** `proposed` · `accepted` · `superseded by ADR-XXXX` · `deprecated`

## Index

- [ADR-0001: Single-file pixel-art frontend](0001-single-file-frontend-pixel-art.md) — accepted
- [ADR-0002: Railway over Vercel](0002-railway-over-vercel.md) — accepted

## How to add

1. Copy an existing ADR, bump the number
2. Fill Context → Decision → Consequences
3. Commit alongside the code change that implements it
